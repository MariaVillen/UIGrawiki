<Meta title="Foundation/icons" />
<h3> Icons </h3>

We can use the Icons using the different icons components, or we can use the Icon Component.

Icon Component allows you to use a react component or an image as an icon.

Exemple:

<div className="gwk-p-6">
  <Icon
    alt="Close this modal"
    elementSrc={CloseRoundedFill}
    className="gwk-text-white gwk-h-8 gwk-w-8 gwk-rounded-full gwk-bg-black"
  />
</div>
<CodeToClipboard
  code={`<Icon alt="Close this modal" elementSrc={CloseRoundedFill} className="gwk-text-white gwk-h-8 gwk-w-8 gwk-rounded-full gwk-bg-black"/>`}
  className="gwk-text-[10px] gwk-bg-gray-200 gwk-text-black gwk-w-[100%] gwk-flex gwk-justify-between gwk-items-center"
  buttonStyles="gwk-text-white gwk-bg-primary hover:gwk-bg-blue-500"
/>

You can also use un image instead. You need to use the url or path to the image:

<div className="gwk-p-6">
  <Icon alt="GrawikiLogo" elementSrc="/vite.svg" className="gwk-h-8 gwk-w-8" />
</div>
<CodeToClipboard
  code={`<Icon alt="GrawikiLogo" elementSrc="../../public/vite.svg" className="gwk-h-8 gwk-w-8" />`}
  className="gwk-text-[10px] gwk-bg-gray-200 gwk-text-black gwk-w-[100%] gwk-flex gwk-justify-between gwk-items-center"
  buttonStyles="gwk-text-white gwk-bg-primary hover:gwk-bg-blue-500"
/>
